## BURST Cross-Platform Wallet UI

According to [CIP-18](https://github.com/burst-apps-team/CIPs/blob/master/cip-0018.md) this is the new Cross-Platform Burst Wallet

Targeted platforms are

- Mac OS
- Linux
- Windows
- Android
- iOS
